## How to Use

### Step 1: Create Campaign
1. Navigate to **Campaigns** ‚Üí **Create Campaign**
2. Fill in basic details (name, audience, etc.)
3. Write your message in the email body field

### Step 2: Open Enhancer
1. Scroll to the **Email Body** section
2. Click the **‚ú® Enhance with AI** button
3. Modal opens with side-by-side view

### Step 3: Select Tone
1. Review the 6 tone options in the grid
2. Click your desired tone button
3. Read the tone description
4. Wait 2-5 seconds for enhancement

### Step 4: Review Enhancement
**Left Panel:** Your original message  
**Right Panel:** AI-enhanced version

**Compare:**
- Length and structure
- Word choice improvements
- Tone appropriateness
- Call-to-action clarity

### Step 5: Try Another Tone (Optional)
1. Click **Try Another Tone** button
2. Select different tone
3. Compare new version
4. Repeat as needed

### Step 6: Apply Enhanced Message
1. Click **Apply Enhanced Message** button
2. Enhanced text replaces original in campaign form
3. Continue editing manually if desired
4. Save campaign

## Edge Cases

### Very Long Messages
**Limit:** 500 tokens (~2000 characters)

If your message is too long:
1. Split into multiple sections
2. Enhance each section separately
3. Combine manually after

**Or:** Shorten original message first, then enhance

---

### Non-English Text
The enhancer works best with English. For other languages:
- Results may vary in quality
- Consider using a translation service first
- Test with small samples

---

### Special Formatting
The AI preserves:
- Paragraph breaks
- Lists (bullet points, numbers)
- Basic structure

May not preserve:
- HTML tags
- Complex formatting
- Custom styling

**Tip:** Add formatting after enhancement

---

selection of premium properties.

## Troubleshooting

### Button Disabled
**Reason:** Message body is empty  
**Solution:** Write some text first, then click Enhance

### Enhancement Taking Too Long
**Reason:** OpenAI API latency  
**Solution:** Wait up to 10 seconds, refresh if needed

### Enhanced Text Seems Off
**Reason:** Unclear original message or wrong tone selected  
**Solution:** 
1. Rewrite original more clearly
2. Try different tone
3. Edit enhanced version manually

### Lost Enhancement After Closing Modal
**Reason:** Didn't click "Apply Enhanced Message"  
**Solution:** Must click Apply button to save, re-enhance if needed

## FAQ

### Q: How much does each enhancement cost?
**A:** Approximately $0.01-0.03 per enhancement (tracked automatically)

### Q: Can I undo an enhancement?
**A:** Not automatically. Use your browser's undo (Ctrl+Z) or keep original text in a separate document.

### Q: Will the AI add fake information?
**A:** No! The AI only rewrites existing content. It doesn't invent facts, prices, or details.

### Q: Can I save favorite enhancements?
**A:** Not currently. Consider creating template campaigns with pre-enhanced messages.

### Q: How do I create custom tones?
**A:** Contact your administrator to add custom tone configurations.

### Q: Is my message data private?
**A:** Yes. Messages are sent to OpenAI API with encryption, not used for training.
